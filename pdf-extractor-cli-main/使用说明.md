# 一键启动使用说明

## 快速开始

### 方式一：双击批处理文件（最简单）

1. **完整功能模式**：双击 `启动.bat`
   - 提供交互式界面，可以选择所有提取选项
   - 适合需要自定义配置的用户

2. **快速提取表格模式**：双击 `快速提取表格.bat`
   - 一键提取PDF中的所有表格
   - 使用默认配置（CSV格式）
   - 适合快速提取财务报表

### 方式二：命令行启动

#### 交互式模式（推荐）
```bash
python start.py
```

#### 快速模式（仅提取表格）
```bash
python start.py --quick
```

#### 快速模式（指定文件）
```bash
python start.py --quick --file "examples/腾讯控股：2019年报.pdf"
```

## 功能说明

### 交互式模式

启动后会依次引导您：

1. **选择PDF文件**
   - 可以直接输入文件路径
   - 也可以拖拽PDF文件到命令行窗口

2. **选择提取选项**
   - 提取文本：提取PDF中的纯文本内容
   - 提取表格：提取PDF中的表格（支持财务报表识别）
   - 提取图片：提取PDF中嵌入的图片
   - OCR识别：对图片型PDF进行OCR文字识别

3. **选择页码范围**（可选）
   - 所有页面：提取整个PDF
   - 指定范围：如 `1-3,5,7-9` 表示第1-3页、第5页、第7-9页

4. **设置输出目录**（可选）
   - 默认：`output/`
   - 可自定义输出路径

5. **选择表格格式**（仅当选择了提取表格时）
   - CSV格式：适合Excel打开
   - JSON格式：适合程序处理

### 快速模式

- 自动提取PDF中的所有表格
- 使用默认配置：
  - 提取所有页面
  - 输出格式：CSV
  - 输出目录：`output/`

## 输出文件结构

```
output/
└── [PDF文件名]/
    ├── csv/          # 表格文件（CSV格式）
    ├── txt/          # 文本文件
    ├── images/       # 图片文件
    └── ...
```

## 使用示例

### 示例1：提取财务报表
1. 双击 `快速提取表格.bat`
2. 输入PDF文件路径（或拖拽文件）
3. 等待提取完成
4. 在 `output/[PDF文件名]/csv/` 目录查看提取的表格

### 示例2：完整提取
1. 双击 `启动.bat`
2. 按提示选择：
   - PDF文件
   - 提取选项（建议选择：文本、表格）
   - 页码范围（默认全部）
   - 输出目录（默认output）
   - 表格格式（默认CSV）
3. 确认配置后开始提取

### 示例3：提取指定页面
1. 运行 `python start.py`
2. 选择PDF文件
3. 选择提取选项
4. 输入页码范围：`1-10`（只提取前10页）
5. 开始提取

## 注意事项

1. **文件路径**：如果路径包含空格，可以使用双引号括起来
2. **拖拽文件**：Windows系统可以直接将PDF文件拖拽到命令行窗口
3. **编码问题**：程序已自动处理UTF-8编码，支持中文文件名和路径
4. **提取质量**：程序针对财务报表做了优化，能自动识别和过滤无效表格

## 常见问题

### Q: 提示"未找到Python"
A: 请确保已安装Python 3.11或更高版本，并且已添加到系统PATH环境变量中

### Q: 没有提取到表格
A: 
- 确认PDF中确实包含表格（非图片形式的表格）
- 尝试使用交互式模式，选择"提取表格"选项
- 查看日志输出了解详情

### Q: 提取的表格不完整
A: 
- 程序会自动过滤低质量的表格
- 如果表格被分割在多页，可能需要手动合并
- 可以尝试调整页码范围重新提取
